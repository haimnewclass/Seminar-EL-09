Student Assignment: Understanding XGBoost Code - 10 Steps

Objective:

This assignment will help you understand how to work with the XGBClassifier using the Iris dataset. You will learn how to install necessary libraries, build and evaluate models, and experiment with model parameters. The steps are structured progressively to help you explore and understand the code step by step.

Step 1: Install Required Libraries

Task:

  • Install the required libraries: xgboost and scikit-learn.
  • Use the following command:
    pip install xgboost scikit-learn

Verify Installation:

  • Run the following code in Python:
    import xgboost
from sklearn.datasets import load_iris
print("Libraries imported successfully!")

Step 2: Load Data

Task:

  • Load the Iris dataset from scikit-learn.
  • Use this code:
    from sklearn.datasets import load_iris
data = load_iris()
print("Feature Names:", data.feature_names)
print("Target Names:", data.target_names)
  • Question for Students:
    • How many features are there in the dataset? What do they represent?

Step 3: Split the Data

Task:

  • Split the dataset into training and testing sets using train_test_split:
    from sklearn.model_selection import train_test_split
X_train, X_test, y_train, y_test = train_test_split(data.data, data.target, test_size=0.2, random_state=42)
print("Training Set Size:", X_train.shape)
print("Test Set Size:", X_test.shape)
  • Question for Students:
    • Why do we split the data? What are the training and testing sets used for?

Step 4: Create an XGBoost Model

Task:

  • Create a basic model using XGBClassifier with the following parameters:
    from xgboost import XGBClassifier
model = XGBClassifier(n_estimators=10, learning_rate=0.1, max_depth=3)
  • Question for Students:
    • What does each parameter (e.g., n_estimators and max_depth) represent?

Step 5: Train the Model

Task:

  • Train the model using the training set:
    model.fit(X_train, y_train)
print("Model trained successfully!")
  • Question for Students:
    • What does the fit method do?

Step 6: Make Predictions

Task:

  • Use the trained model to predict results for the testing set:
    y_pred = model.predict(X_test)
print("Predictions:", y_pred)
  • Question for Students:
    • What is the output of the predict method?

Step 7: Evaluate the Model

Task:

  • Evaluate the accuracy of the model using accuracy_score:
    from sklearn.metrics import accuracy_score
accuracy = accuracy_score(y_test, y_pred)
print("Accuracy:", accuracy)
  • Question for Students:
    • What does the accuracy metric represent?

Step 8: Predict a Single Record

Task:

  • Predict the class for a new record (e.g., [5.1, 3.5, 1.4, 0.2]):
    test_record = [[5.1, 3.5, 1.4, 0.2]]
prediction = model.predict(test_record)
print("Prediction for test record:", prediction)
  • Question for Students:
    • What is the purpose of the predict method in this case?

Step 9: Explore the Test Set

Task:

  • Predict the first 10 examples from the testing set and compare them to the actual values:
    test_records = X_test[:10]
predictions = model.predict(test_records)
for i in range(10):
    print(f"Record {i+1}: Predicted = {predictions[i]}, Actual = {y_test[i]}")
  • Question for Students:
    • What can you infer about the model's accuracy based on this comparison?

Step 10: Experiment with Parameters

Task:

  • Experiment with the model's parameters (n_estimators, learning_rate, max_depth) and observe how they affect the accuracy:
    model = XGBClassifier(n_estimators=50, learning_rate=0.05, max_depth=4)
model.fit(X_train, y_train)
y_pred = model.predict(X_test)
print("New Accuracy:", accuracy_score(y_test, y_pred))
  • Question for Students:
    • How does changing the parameters impact the accuracy? What happens when you adjust learning_rate or max_depth?

Assignment Summary:

By completing this assignment, students will understand:

  1. How to use XGBClassifier effectively.
  2. How to evaluate model performance.
  3. How to tweak parameters to improve model accuracy.

Bonus for Advanced Students:

  • Use the feature importance plot to see which features are most significant in the model:
    from xgboost import plot_importance
import matplotlib.pyplot as plt
plot_importance(model)
plt.show()
